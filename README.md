## 本地运行demo
- 确定本机安装了`nodejs`，可使用`node -v`验证
- 下载源码，解压，或者`git clone https://github.com/kaiyu5609/kulp.git`
- 进入源码目录，找到`gulpfile.js`命令行中运行`gulp serve`
- 打开浏览器访问 `http://localhost:5000/example/index.html`
